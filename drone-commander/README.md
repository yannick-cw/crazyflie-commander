# drone-commander

A terminal ground station for a [Crazyflie][] nano-drone. Selects a saved
mission, runs it, and shows live telemetry (position, velocity, battery). Built
on [ratatea](../ratatea).

> Work in progress.

[Crazyflie]: https://www.bitcraze.io/products/crazyflie-2-1-plus/
